Required Libraries

Ensure you have the following libraries installed before using Minetroller:

  • cv2
  • mediapipe
  • pyautogui
  • torch
  • torch.nn
  • torch.optim
  • numpy
  • json
  • tkinter
  • sklearn.preprocessing
  • sklearn.decomposition
  • threading
  • multiprocessing
  • logging
  • os
  • time
  • PIL

You can install these using pip:

pip install opencv-python mediapipe pyautogui torch scikit-learn pillow

About Minetroller

Minetroller is a revolutionary application that lets you control Minecraft with real-life movements! It uses advanced pose detection and machine learning techniques to interpret your gestures into Minecraft actions.
